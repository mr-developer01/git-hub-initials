--- git init

# Stages of any changes in our Local git repo :--
U - untracked
A - Added or staged
C - Commited

# Making a checkpoint or saved point :--
Adding a file
Staging a file
Commiting a file

# Checking saved point whatever made while change in codebase :--
--- git log --oneline
